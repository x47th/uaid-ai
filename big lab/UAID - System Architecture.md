## Data Flow

1. **Obsidian** вҶҗ Manual knowledge entry, planning, tracking
2. **Neo4j** вҶҗ Structured relationships (Client вҶ’ Problem вҶ’ Solution)
3. **Qdrant** вҶҗ Semantic document vectors (1024-dim)
4. **GraphRAG** вҶҗ Queries Neo4j + Qdrant, augments DeepSeek prompts
5. **CrewAI** вҶҗ 3-agent pipeline: Research вҶ’ Architect вҶ’ Write
6. **GraphAI** вҶҗ Declarative workflow automation

## Key Design Decisions

- **DeepSeek V4 Flash** as primary LLM (cost-effective, strong reasoning)
- **Neo4j Community Edition** вҖ” free, runs in Docker
- **Qdrant** вҖ” lightweight vector DB, single binary
- **CrewAI sequential process** вҖ” researcher вҶ’ architect вҶ’ writer
- **Pseudo-embeddings** currently (hash-based), should upgrade to DeepSeek embeddings or sentence-transformers
